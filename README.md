# bitbar

1. clone in your bitbar plugin directory
1. `brew install python3`
1. `pip3 install json locale pyyaml`
1. create .env.yaml file

```yaml

eth:
  quantity: 1
  eurQty: 9999

xrp:
  quantity: 1+1
  eurQty: 999+1

xem:
  quantity: 1
  eurQty:  999999

xlm:
  quantity: 1
  eurQty: 9999

bcn:
  quantity: 1
  eurQty: 9999
  
```
